SETUP INSTRUCTIONS:
    1. Install python3.8 for your platform.
    2. Install required python packages using pip install -r requirements.txt command.
    3. Create a new bot and add it to your server.
    4. Enter your plex server login details and your bot token in config.json file. 
    NOTE : YOU NEED TO RENAME THE sample-config.json FILE TO config.json.
    5. Run the bot using python3 main.py command. 
    6. Use help command to see the available commands.
